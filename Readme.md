# MtcBot

MtcBot เป็นโปรเจกต์บอทที่ผมสร้างขึ้นเพื่อทดลองและเรียนรู้การพัฒนาบอทด้วยภาษา Python  
โปรเจกต์นี้เป็นส่วนหนึ่งของการฝึกเขียนโค้ดจริง ตั้งแต่การจัดโครงสร้างไฟล์ การแยก logic  
ไปจนถึงการทำให้บอทสามารถตอบสนองคำสั่งได้ตามที่ออกแบบไว้

โปรเจกต์นี้ไม่ได้เน้นความสมบูรณ์แบบ แต่เน้นการเรียนรู้และพัฒนาทักษะของตัวเอง

---

## สิ่งที่บอททำได้
- รับและประมวลผลคำสั่งจากผู้ใช้
- แยกการทำงานเป็นโมดูล (handlers / features)
- จัดการ logic พื้นฐานของบอท
- เป็นฐานสำหรับต่อยอดฟีเจอร์อื่นในอนาคต

(ฟีเจอร์อาจเปลี่ยนแปลงหรือเพิ่มขึ้นได้เรื่อย ๆ ตามการพัฒนา)

---

## ทำไมถึงสร้างโปรเจกต์นี้
ผมสร้าง MtcBot ขึ้นมาเพื่อ:
- ฝึกเขียน Python จากการทำโปรเจกต์จริง
- เรียนรู้การจัดโครงสร้างโค้ดให้เป็นระเบียบ
- ทดลองสร้างบอทที่สามารถนำไปต่อยอดได้ในอนาคต
- ใช้เป็น portfolio และบันทึกพัฒนาการของตัวเอง

---

## เทคโนโลยีที่ใช้
- Python
- Libraries ที่จำเป็นสำหรับการทำงานของบอท (ดูใน `requirements.txt`)

---

## หมายเหตุเกี่ยวกับโค้ด
โปรเจกต์นี้มีการใช้ AI เป็นผู้ช่วยในการเขียนโค้ดบางส่วน  
แต่ผมอ่าน ทำความเข้าใจ และปรับแก้โค้ดด้วยตัวเอง  
เป้าหมายคือการเรียนรู้ ไม่ใช่แค่ให้โค้ดรันได้อย่างเดียว

---

## สถานะโปรเจกต์
โปรเจกต์นี้ยังอยู่ในระหว่างการพัฒนาและปรับปรุง  
โค้ดอาจมีการเปลี่ยนแปลงหรือ refactor ได้ตลอดเวลา

---

## ผู้พัฒนา
พัฒนาโดย M1N-W  
โปรเจกต์นี้สร้างขึ้นเพื่อการเรียนรู้และพัฒนาทักษะส่วนตัว
